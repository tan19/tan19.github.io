# Question
Prove that $E[B_t^{2k}] = \frac{(2k)!}{2^k\dot k!} t^k$, where $B_t$ is a Brownian motion.

# Answer
Recall that the $n^{th}$ centralized moment of a standard normal random variable is
\begin{align}
    E[Z^n] = (n-1)!!
\end{align}
Therefore,
\begin{align}
    E[B_t^{2k}] = t^k (2k-1)!! = \frac{(2k)!}{2^k k!} t^k
\end{align}

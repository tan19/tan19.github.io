# Question
Which number is larger, $\pi^e$ or $e^\pi$?

# Answer
Take log on both sides we have $\frac{\log \pi}{\pi} < \frac{\log e}{e}$ because $\left(\frac{\log x}{x}\right)' = - \frac{\log x + 1}{x^2} < 0$ for $x > e$. Therefore, $e^\pi > \pi^e$.

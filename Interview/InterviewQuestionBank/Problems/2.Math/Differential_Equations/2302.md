# Question
Solve $y'(x) + ay(x) = f(x)$ with $f(0)=x_0$.

# Answer
This is a first-order linear ODE and the integrating factor is $I(x) = e^{\int a dx} = e^{ax}$. Multiplying both sides by $e^{ax}$ we obtain $y^2 = 2 \cdot \frac{\int f(x) e^{ax} dx}{e^{ax}} + C$.

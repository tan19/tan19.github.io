# Question
Solve $y' = \frac{x-y}{x+y}$.

# Answer
Let $z = x+y$, then the problem becomes $y' = \frac{2x}{z} - 1$, which is a separable differential equation. The answer is $y^2 + 2xy -x^2 = C$.

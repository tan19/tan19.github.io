# Question
Explain Newton's method.

# Answer
Suppose we have a function $f(x) = 0$ and we want to find its root. We start with $x_0$ close to its root, and update as
\begin{align}
    x_{n+1} = x_n - \frac{f(x_n)}{f'(x_n)}
\end{align}

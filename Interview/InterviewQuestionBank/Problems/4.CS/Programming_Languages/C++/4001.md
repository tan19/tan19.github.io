# Question
What's wrong with the following code?

```
class A {
private:
    int value;
public:
    A(int n) { value = n; }
    A(A other) { value = other.value; }

    void Print() {std::cout << value << std::endl; }
};

int _tmain(int argc, _TCHAR* argv[]) {
    A a = 10;
    A b = a;
    b.Print();

    return 0;
}
```

# Answer
Should be
```
A(const A& other)
```

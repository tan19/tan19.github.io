2.2 Metric Spaces
2.2.1 Deﬁnitions
Deﬁnition 2.2.1 A set X, whose elements we shall call points, is said to be
a metric space if with any two points p and q of X there is associated a real
number d(p, q), called the distance from p to q, such that
(a) d(p, q) > 0 if p 6= q; d(p, q) = 0;
(b) d(p, q) = d(q, p);
(c) d(p, q) ≤ d(p, r) + d(r, q), for any r ∈ X.
Any function with these three properties is called a distance function, or a
metric.
